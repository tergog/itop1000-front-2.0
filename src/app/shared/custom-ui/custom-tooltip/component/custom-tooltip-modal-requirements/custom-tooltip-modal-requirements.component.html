Email address validation requirements

  • Email name up to @ (test) can consist of 6–30 digits, contain letters (a–z), numbers (0–9) and dots (.).
  • Symbol (@);
  • Domain server name (mail) — one or many symbols; Dot (.).
  • Domain name of firts level (ru) must be from 2 up to 5 letters.
  • Impossible to use ampersand(&), signs of parity (=) and addings (+), brackets (<>), comma (,), underline symbol (_), apostrophe ('), dash (-) and a few dots in a row.
  • Users name can start and end by any permitted symbol except dot (.).
